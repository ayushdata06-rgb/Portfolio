# Parallax Void Portfolio

A modern 3D portfolio scaffold built with Next.js App Router and TypeScript.

## Stack

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- Three.js, React Three Fiber, Drei
- Framer Motion 

## Core Structure

- `app/page.tsx`: page composition and section layout.
- `components/Canvas3D.tsx`: 3D canvas scene with lights and rotating geometry.
- `components/HeroSection.tsx`: animated hero intro using Framer Motion.
- `components/ProjectsSection.tsx`: project showcase placeholders.
- `.vscode/mcp.json`: MCP server configuration for local filesystem and GitHub.
